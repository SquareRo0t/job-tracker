# Jobbansöknings-tracker

En webbapp för att hålla koll på jobb du har ansökt till.

## Funktioner
- Lägg till nya jobbansökningar
- Uppdatera status (Sökt, Intervju, Avslag)
- Ta bort ansökningar

## Tekniker
- Python
- Flask
- SQLite
- HTML/CSS

## Hur man kör projektet

1. Installera Flask:
   pip install flask

2. Starta appen:
   python app.py

3.  Öppna webbläsaren och gå till:
    http://127.0.0.1:5000
